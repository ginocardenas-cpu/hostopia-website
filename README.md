# Hostopia website

Next.js site for Hostopia. This is the **only** project to edit; it deploys to Vercel.

- **Preview / live:** https://hostopia-website-2-jhmfeodju-ginos-projects-21427f6a.vercel.app
- **Repo:** https://github.com/rollen77-spec/hostopia-website

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Push to `main` (or your connected branch) to trigger a Vercel deployment.
